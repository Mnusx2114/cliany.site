# 获取帮助 / Getting Help

欢迎使用 cliany-site！如果您在使用过程中遇到问题或有建议，我们很乐意为您提供帮助。请按照以下步骤获取支持。

## 先查阅文档 / Check Documentation First

在提交问题之前，请先查阅以下资源：

- **官方文档网站**：https://www.cliany.site  
  包含详细的使用指南、API 参考和常见问题解答。

- **README.md 文件**：  
  请查看「使用示例」和「命令参考」章节，了解基本用法和命令说明。

## 提交 Issue / Filing an Issue

如果文档无法解决问题，请在 GitHub 上提交 Issue。我们鼓励社区参与，共同改进项目。

### Bug 报告 / Bug Reports
- 使用 Bug Report 模板描述问题
- 提供详细的复现步骤、环境信息和错误日志

### 功能建议 / Feature Requests  
- 使用 Feature Request 模板提出新功能想法
- 描述您的使用场景和预期效果

**Issues 入口**：https://github.com/pearjelly/cliany.site/issues

## 搜索已有讨论 / Search Existing Issues

提交新 Issue 前，请先搜索是否已有相关讨论。这有助于避免重复，并可能找到现有解决方案。

## 报告安全漏洞 / Reporting Security Vulnerabilities

如果您发现安全漏洞，请**不要在公开 Issue 中报告**。请查看 [SECURITY.md](SECURITY.md) 文件了解安全报告流程。

感谢您的反馈！您的贡献让 cliany-site 变得更好。
